# Trabajo-Final-An-lisis-de-Variaci-n-Gen-mica-en-E.-Uni63
Este repositorio contiene la solución desarrollada para la asignatura de Principios de Programación en Bioinformática. El proyecto consiste en el diseño de un pipeline automatizado en Bash para el procesamiento, filtrado y análisis estadístico de variantes genómicas en formato VCF.
